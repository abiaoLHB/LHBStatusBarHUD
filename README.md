# LHBStatusBarHUD
一个好用的状态栏指示器
##显示成功信息
```objc
[LHBStatusBarHUD LHB_showSuccess:@"加载成功"];
```
##显示失败信息
```objc
[LHBStatusBarHUD LHB_showError:@"加载失败"];
```
##显示普通信息
```objc
    [LHBStatusBarHUD LHB_showNormal:@"普通信息"];
```
##显示正在加载信息
```objc
    [LHBStatusBarHUD LHB_showLoading:@"正在加载"];
```
